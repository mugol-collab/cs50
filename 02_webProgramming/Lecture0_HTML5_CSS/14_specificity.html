The Specificity makes reference to the precedence of operators in CSS
  1. Inline
  2. id
  3. class
  4. type
  1. Inline

    2. Adding the style directly in the html element. Ex: <h4 style="color: blue">Test</h4>

  2. id

    3. Is an unique identifier of an element

  3. class

    4. Is a group identifier of elements

  4. type

    5. type of html element. Ex: <headings>, <lists>, <tables>, etc
